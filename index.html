hello world 깃 커밋 방법 0. git config user.name "계정아이디" git config user.email "계정이메일" 1. 프로젝트 작업 후(디렉토리 추가 / 파일 추가 / 수정 / 삭제...) 2. 로그인 확인 (계정명 확인) gh auth status 3. 변경된 대상 모든 파일들을 모두 스테이징 git add --all 4. 커밋 (변경 내역을 제 3자가 알아볼 수 있는 규칙으로 메세지 입력) Create Add Update Delete ... git commit -m "메세지 입력" 5. 푸시 (로컬 프로젝트를 ☞ 원격 저장소에 반영) git push -u origin main
